# E2E-тесты (QA-02)

Сквозные тесты на Playwright, проверяющие критический пользовательский
сценарий: **загрузка документа → индексация → поиск → отображение
результатов**.

## Предварительные требования

1. Запущен весь стек приложения:
   ```bash
   docker compose up --build
   ```
   Фронтенд должен быть доступен на http://localhost:3000.

2. Сгенерирована тестовая фикстура (используется при загрузке):
   ```bash
   cd backend/tests/fixtures
   python generate_fixtures.py
   ```

## Запуск

```bash
cd e2e
npm install
npx playwright install --with-deps chromium
npm test                 # прогон тестов
npm run test:headed      # с видимым браузером
npm run report           # открыть HTML-отчёт
```

Адрес фронтенда можно переопределить:

```bash
BASE_URL=http://localhost:8080 npm test
```

## Что проверяется

| Сценарий | Требования |
|----------|-----------|
| Загрузка файла и доведение статуса до «Готово» | FE-01, FE-02, FE-03 |
| Поиск по запросу и вывод карточек с полями | FE-04, FE-05 |
| Сообщение при отсутствии результатов | FE-08 |
